Como usar suas chaves API

Autenticação via Header

Authorization: Bearer med_abc123...

Ou usando o header alternativo:

X-API-Key: med_abc123...

Exemplo de requisição

curl https://mediddata.com/api/medicamentos?q=dipirona \
  -H "Authorization: Bearer med_abc123..."

Boas práticas de segurança

  • Nunca compartilhe suas chaves de acesso
  • Revogue imediatamente chaves comprometidas
  • Use chaves diferentes para ambientes dev/prod
  • As chaves respeitam os limites do seu plano
